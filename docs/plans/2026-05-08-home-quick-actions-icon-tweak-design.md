# 首页快捷入口图标微调设计

## 背景

首页“今晚想从哪里开始”区域目前有两个视觉点需要微调：

- 右上角的“3 个核心入口”标签过于像说明性标记，干扰卡片本身的阅读重心。
- 三张快捷入口卡片左上角当前使用单字徽记，识别效率偏低，用户希望替换成更直观的 emoji 图标。

## 目标

- 去掉右上角“3 个核心入口”标签。
- 将三张入口卡片左上角的单字徽记替换为 emoji 图标。
- 保持卡片布局、文案、跳转和现有绘本化风格不变。

## 设计方案

- `创建故事` 使用 `✨`
- `故事库` 使用 `📚`
- `社区分享` 使用 `🌟`

## 非目标

- 不改动首页其他模块。
- 不改动卡片跳转逻辑。
- 不改动 spec 中的核心需求描述，仅做视觉微调。

## 验收标准

- 首页入口区不再显示“3 个核心入口”标签。
- 三张快捷入口卡片左上角显示 emoji 图标。
- 首页现有测试与构建不受影响。
